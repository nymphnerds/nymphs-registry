# Nymphs Registry

This repo is the trusted catalog for NymphsCore Manager.

It does not contain module payloads. It only tells the Manager which Nymph modules are official/trusted and where their `nymph.json` files live.

Plain English:

```text
nymphs-registry = catalog
module repo     = actual module
```

First test module:

```text
github.com/nymphnerds/worbi
```

Registry file:

```text
nymphs.json
```
